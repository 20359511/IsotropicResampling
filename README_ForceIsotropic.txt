README_ForceIsotropic

How to use in Slicer:
Save as a python file on your computer
Open Slicer
Select Module Settings for the Adit -> Application Setting Dialog
Open the side panel and click Add
Add the path to the directory containing ForceIsotropic.py
	(when selecting the directory, the ForceIsotropic.py file itself will not be displayed)
Restart Slicer when prompted. 
Force Isotropic  is now in the Modules Menu, under the category Examples
